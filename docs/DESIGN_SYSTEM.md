# Design System

## Concept

> Bloomberg terminal meets boutique finance consultancy meets modern AI workspace.

## Feel

Premium, spacious, confident, editorial, finance-led, data-rich, calm, practical, industry-specific.

## Colours

| Token | Hex | Use |
|---|---|---|
| Graphite | #1A1D21 | Primary dark background |
| Zinc | #2A2E33 | Panel background |
| Off-white | #F5F4F1 | Light background / text on dark |
| Muted green | #4C6B57 | Positive KPI, accents |
| Dark slate | #10131A | Deepest background |
| Copper | #B5714A | Primary accent, CTAs |
| Amber | #C99A3D | Warning states |
| Blue-grey | #7C8A99 | Secondary data / muted text |

## Typography

- Headings: a confident serif or high-contrast grotesk (system fallback: Georgia / Inter).
- Body: Inter or system-ui, generous line height for data-heavy pages.
- Numerals: tabular figures for all KPI/financial data.

## Visual motifs

- KPI cards with clear delta indicators
- Data panels with subtle borders, not heavy shadows
- Grid backgrounds (fine, low-contrast)
- Stock/transport movement line charts
- Understated industrial texture (fine hairline grids, not photos of scrap yards)
- Premium glass panels (subtle blur, low opacity) on dark sections
- Clean dashboards, tabular data, real numbers
- No cheap recycling icons, no cartoon scrap imagery

## What the design communicates

Trust, clarity, specialist knowledge, operational control, finance discipline, modern reporting.

## Component principles

- `KpiCard`: value, label, trend delta, optional sparkline.
- `PremiumCard`: glass-panel style container for feature/proof blocks.
- `ServiceCard` / `PricingCard`: consistent card grid, copper accent on primary tier.
- Section headings: small eyebrow label + large heading + short supporting line.
