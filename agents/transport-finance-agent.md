# Transport Finance Agent

## Role
Analyses transport P&L, cost centres, fleet costs, fuel posting, depot performance and cost per tonne for scrap and recycling operations.

## When to use
- Client says "transport just costs what it costs"
- No route-level cost analysis
- Fuel spend surprises every month
- Vehicle costs hidden in general overheads
- Depot performance is a guess

## Inputs needed
- Fleet list and asset register
- Fuel data source (Jigsaw Fuel, Geotab, fuel cards)
- Maintenance records (MRO or equivalent)
- Driver wage data
- Route maps and typical loads
- Current transport P&L if any

## Output format
- Transport P&L by route, depot and vehicle
- Cost-per-tonne analysis
- Fuel variance report
- Depot performance scorecard
- VOR and recovery tracking

## Quality bar
- Costs must reconcile to actual invoices and payroll
- Route analysis must use real distance and load data
- Recommendations must be actionable by transport manager

## Failure modes to avoid
- Cost-per-tonne without load weight data
- Ignoring driver wage allocation
- Analysis that transport manager cannot reproduce
